First time? Click here for setup instructions
▼

You only need to do steps 1–3 once ever.
Step 4 you do each time you want to use this page.

1
Open PowerShell
Press the Windows key on your keyboard (bottom-left, looks like ⊞).
Type PowerShell and press Enter.
A blue or black window will open — that's normal, don't worry about it.
2
Install Bun — only once
Click Copy on the command below, then click inside the PowerShell window and right-click → Paste, then press Enter.
Wait until the text stops scrolling and you can type again.
powershell -c "irm bun.sh/install.ps1 | iex"
⚠ When finished: close PowerShell completely, then reopen it (repeat step 1 again). This is required.
3
Install the compressor — only once
In the freshly reopened PowerShell window, click Copy below, paste it in, and press Enter.
bun i -g glb-compressor
Wait for it to finish. Some text will scroll — that's normal. When the cursor comes back, it's done.
4
Start the server — every time you use this page
Open PowerShell (step 1), then paste and run this:
glb-server
You'll see something like "Listening on port 8080" — that means it's ready.
⚠ Keep PowerShell open! Do not close that window while using this page. The compressor runs through it. If you close it by accident, just reopen PowerShell and run glb-server again.
